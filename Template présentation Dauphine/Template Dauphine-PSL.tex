\begin{frame}{Modèle Mathématique}
  L'équation principale de notre modèle est :
  \begin{equation}
    E = mc^2 \label{eq:einstein}
  \end{equation}

  Où :
  \begin{itemize}
    \item $E$ représente l'énergie
    \item $m$ est la masse
    \item $c$ est la vitesse de la lumière dans le vide
  \end{itemize}

  Nous faisons référence à l'équation \eqref{eq:einstein}.

  Une autre équation importante :
  $$ \sum_{i=1}^{n} x_i = x_1 + x_2 + \dots + x_n $$
\end{frame}
